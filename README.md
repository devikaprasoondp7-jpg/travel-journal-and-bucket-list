✈️ Travel Journal and Bucket List Portal
A web-based application designed to help users plan their travels, manage a bucket list, and journal their experiences with customizable themes.

🚀 Features
User Authentication: Secure Login and Registration system using localStorage.

Travel Preferences: Users can select preferences (Nature, Food, City, Beach) during registration.

Theme Customization: Switch between Pink, Blue, and Green themes which update the UI immediately.

Dynamic Background: Interactive background featuring floating triangles and circles that match the selected theme.

Data Persistence: Uses localStorage to save user credentials and preferences across sessions.

🛠️ Tech Stack
HTML5: Structure of the application.

CSS3: Styling, layout, and theme variables.

JavaScript (Vanilla): Logic for authentication, theme switching, and local storage management.

⚙️ Installation & Running
Clone this repository to your local machine.

Open index.html in your web browser.

Register a new account to get started.

📝 How to use
Login/Register: Enter a username and password. If you don't have one, click "Register".

Select Preferences: (New Users) Check your travel interests.

Choose Theme: Use the dropdown menu to select your favorite theme color.

Login: Click "Login" to access the main travel portal.
